# Waterloodle
### A UWaterloo campus daily game
EngHacks 2026 Submission by Jack Gordon (jgordon1482@gmail.com), Charles Huang (charles6huang@gmail.com), Mya Kay (myakay38@gmail.com) and Jasmine Yip (JasmineY05)
### HOW TO RUN:
1. Make sure you have node and npm
2. In the waterloodle folder, run:
```
npm install
npm start
```
3. Open the link given in the command line

### Citations
Generative AI including Grok, ChatGPT, Gemini were used for minor code creation and debugging. Comments were left inside files where notable AI generation was used.

As well, the building information blurb shown on the end screen was mostly generated using ChatGPT, though it was all reviewed by a human reviewer.

Data was also taken from the official UWaterloo map site: https://uwaterloo.ca/map/ as a basis for our map and information.
